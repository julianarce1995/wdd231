Movie Hub Explorer - Site Plan

Planning document for the Week 5 project website.

Site Name

Movie Hub Explorer

This name was selected because the site is a central place (hub) where users can explore trending movies, discover new titles, and review detailed movie information.

Optional domain availability: moviehubexplorer.com (representative example)

Site Purpose

The site provides a focused and visual movie discovery experience. It combines data from an external movie API and a local JSON source to present trending content, curated movie lists, and detailed pages with synopsis, release date, and ratings.

Scenarios

  • What movies are trending right now that I can watch this weekend?
  • How can I quickly find the release date, rating, and summary for a specific movie?
  • Which movies in the Discover section are locally curated recommendations?

Color Scheme

This document uses the selected palette exclusively.

#0F4C81

Primary Blue: headings, header/footer background, section accents, and key structure elements.

#F4B400

Accent Gold: emphasis highlights, border accent line, and callout details.

Support neutrals: light background and dark body text to preserve readability and visual balance.

Typography

  • Bebas Neue: used for page headings and section titles to create a cinematic visual voice.
  • Source Sans 3: used for all body content and lists for high readability.
  • Fira Code: used only for wireframe labels and structural ASCII sketches.
